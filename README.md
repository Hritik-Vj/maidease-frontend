# MaidEase Frontend (React PWA)

The MaidEase frontend is a React-based Progressive Web App (PWA) designed to give customers and maids a clean, simple, and mobile-friendly experience.  
It talks to the backend through REST APIs and uses modern UI patterns, offline caching, and fast page loads thanks to Vite + PWA support.

---

## Tech Stack

- React (Vite)
- Redux Toolkit (state management)
- React Router
- Axios (API calls)
- PWA
- Jest + React Testing Library (tests)
